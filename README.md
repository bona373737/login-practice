# login-practice

## 프로젝트 개발환경
1. 코드일관성 유지
  - ESLint, Prettier, husky
  - 변수,함수명 camel case적용
2. 깃브랜치 관리
```
  main ─ develop ─┬─ feature/session   
                  ├─ feature/jwt   
                  └─ feature/oauth   
```
3. 프레임워크,빌드툴
  - React.js
  - TypeScript
  - Vite
4. 버전관리
  - npm ci로 패키지 설치
  - .npmrc파일생성, package.json에 engines 속성 추가
